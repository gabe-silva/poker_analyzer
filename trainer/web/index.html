What The Platform Does

  • Parse PokerNow JSON hand histories in bulk.
  • Bucket players by stable PokerNow player ID while preserving all observed display names.
  • Build fast profile stats dashboards with optional exploit recommendations.
  • Deliver style-based training and live friend-vs training at higher tiers.

Screenshot Setup Checklist

  1. Capture your UI screenshots locally and name them exactly as shown above.
  2. Copy them into trainer/web/screenshots/.
  3. Commit and push to GitHub, then pull on PythonAnywhere and reload the web app.
# local (VS Code terminal)
cd /Users/gabe/Desktop/poker_analyzer
git add trainer/web/index.html trainer/web/styles.css trainer/web/screenshots/
git commit -m "Update tier messaging and landing screenshot gallery"
git push origin main

# PythonAnywhere bash
workon poker-analyzer
cd ~/poker_analyzer
git pull origin main
# then click "Reload" on the PythonAnywhere Web tab

Tier Boundaries

  • Free: analyzer only, one profile bucket, no exploit panel.
  • Pro: analyzer + exploit panel + style-based training in Setup/Trainer.
  • Elite: friend-specific training (live/drill) unlocked on Play page.
